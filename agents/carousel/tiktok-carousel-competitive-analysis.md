### Top-Performing Carousel Patterns in Automation Niche

#### Hook Patterns That Go Viral (100K+ views):

1. **"[Number] AI tools that will [impossible-sounding benefit]"**
   - Example: "7 AI tools that will save you 40 hours this week"
   - Why: Specific number + specific benefit + slight impossibility
   - Performance: Consistently 200K-1M+ views

2. **"I replaced [expensive thing] with [this automation]"**
   - Example: "I replaced my $3K/mo VA with this n8n workflow"
   - Why: Cost savings = universal hook, creates curiosity
   - Performance: 100K-500K views

3. **"Stop using [popular tool]. Use [this] instead."**
   - Example: "Stop manually posting content. This workflow does it for you."
   - Why: Pattern interrupt + contrarian positioning
   - Performance: 150K-400K views

4. **"This [one workflow/tool] does what 5 people used to do"**
   - Why: Dramatic comparison, specific claim
   - Performance: 100K-300K views

5. **"The $0 automation that makes me $[X]K/mo"**
   - Why: Free + revenue = irresistible combination
   - Performance: 200K-800K views

#### Slide Structure Patterns (What's Actually Going Viral):

**DOMINANT STRUCTURE: HOOK â†’ LIST/TIPS (4-8 slides) â†’ CTA**
- Most viral automation carousels are **NOT using a 6-slide narrative arc**
- Instead, they use a **listicle format: 7-10 slides**
- Slide 1: Hook (bold text, dark background or AI-generated scene)
- Slides 2-8: One tool/tip/step per slide with screenshot or clean graphic
- Final slide: CTA ("Follow for more" or "Save this for later")
- **Average viral carousel: 7-9 slides** (not 6)

**SECONDARY STRUCTURE: TUTORIAL/WALKTHROUGH**
- Slide 1: Hook ("Here's how I automated my entire [process]")
- Slide 2: Overview/context
- Slides 3-6: Step-by-step screenshots of actual workflow
- Slide 7: Result/proof
- Slide 8: CTA
- This is closer to Duncan's proposed structure but uses more slides

**TERTIARY STRUCTURE: BEFORE/AFTER TRANSFORMATION**
- Slide 1: Hook with result
- Slide 2: Before state (ugly spreadsheet, manual process)
- Slide 3: The "aha moment" / catalyst
- Slide 4-5: The system/workflow screenshots
- Slide 6: After state (clean dashboard, metrics)
- Slide 7: CTA

### What Makes Automation Carousels Save-Worthy:
1. **Specific tool names + use cases** (not vague "AI tools")
2. **Actual screenshots** of workflows, not just descriptions
3. **Step-by-step replicability** â€” viewers save when they want to do it themselves later
4. **Cost/time comparisons** â€” "$0 vs $3K/mo" drives saves
5. **"Bookmark this"** â€” explicit save CTA works better than "follow"

### What Drives Swipe-Through Completion:
1. **Numbered slides** ("1/7", "2/7") â€” creates progress bar psychology
2. **Each slide reveals new tool/step** â€” curiosity gap between slides
3. **Progressive reveal** â€” build from simple to advanced
4. **Cliffhanger text** at bottom of slide ("but the next one is the real game-changer")
5. **Consistent visual template** â€” reduces cognitive load, trained swipe behavior
